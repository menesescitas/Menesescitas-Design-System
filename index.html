06 — Motion

Animación

6 duraciones × 5 easings. Spring para elementos que aparecen. Snappy para navegación. Instant solo para feedback táctil.

Guía de uso
instant 50ms — Focus ring, checkbox tick
fast 100ms — Hover states, button active
normal 200ms — Tooltips, dropdowns, tabs
slow 350ms — Modals, page transitions
slower 500ms — Hero animations
crawl 800ms — Dramatic intros, loading
⚠ prefers-reduced-motion: Todas las animaciones se reducen a 0.01ms. Respeto WCAG 2.1 SC 2.3.3.

• role="status" para toasts dinámicos
• aria-live="polite" para actualizaciones
08 — Handoff

Figma Guide

Especificaciones para implementar el design system en Figma con Variables, Styles y componentes auto-layout.

🎨
Color Styles → Variables
Crear colección "Menesescitas" con grupos: color/primary/*, color/accent/*, color/semantic/*. Usar Variable Modes para Light/Dark. Nombrar variables exactamente como los tokens JSON.
Aa
Text Styles
9 estilos de texto: Display/Hero, Display/Regular, Heading/H1…H4, Body/LG, Body/MD, Caption. Bind a Typography Variables cuando Figma lo permita.
⚙️
Componentes Auto Layout
Todos los componentes usan Auto Layout con Spacing tokens vinculados. Propiedades expuestas en Panel: Variant (primary/secondary/ghost), Size (sm/md/lg), State (default/hover/disabled).
📦
Tokens Studio Plugin
Importar menesescitas-tokens.json directamente via Tokens Studio para Figma. Sync bidireccional con GitHub para design-dev handoff.
